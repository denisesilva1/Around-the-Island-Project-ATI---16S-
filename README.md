# Around-the-Island-Project-ATI---16S-
The Around the Island (ATI) 16S project aims to characterize the diversity and composition of microbial communities in coral reef environments. Environmental DNA (eDNA) samples were collected from 200 sites around the island of Mo’orea (French Polynesia), spanning five reef habitats: bays, mid-lagoon, fringing reefs, reef crest, and reef passes.
